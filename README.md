## Technologies
The API is created with:
* NodeJS
* Express
* Jasmine - Unit tests

## Setup
npm install express
sudo apt install node-jasmine

## Usage

JSON format Input parameters
{
	"datetime1": "1987-07-13T16:14:18",
	"zone1":  "Asia/Anadyr",
	"datetime2":  "2022-10-02T23:13:13",
	"zone2": "America/Bogota",
	"option": "h"
}

JSON format request
{
	"days": 12866,
	"weekdays": 9190,
	"completeWeeks": 1837,
	"result": "308784 hours"
}
